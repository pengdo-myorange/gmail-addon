# 보내기 전에

Gmail 발송 전 한국어 비즈니스 이메일의 오류를 잡아내고 수정해주는 Chrome 확장기능.

## 주요 기능

- Gmail 작성 창에 **[검토하기]** 버튼 추가
- 9개 카테고리 오류 감지: 수신자 호칭, 중복 표현, 띄어쓰기, 오타, 경어체, 누락 요소, 어색한 표현, 조사 오류, 문단 구분
- Google Gemini API를 활용한 실시간 검토
- 수정된 전체 이메일을 모달로 미리보기 후 한 번에 반영
- 작성자 문체 유지

### v1.1 추가 기능

- **카테고리 토글**: 설정에서 원하는 검토 카테고리만 선택
- **사용자 규칙**: 추가 검토 규칙을 직접 지정
- **모델 선택**: Gemini 2.0 Flash / Flash Lite / 2.5 Flash 중 선택
- **TOP 3 인사이트**: 자주 하는 실수를 검토 결과에 표시

## 설치 방법

### 개발자 모드 설치

1. 이 저장소를 클론합니다
2. Chrome에서 `chrome://extensions` 접속
3. "개발자 모드" 활성화
4. "압축 해제된 확장 프로그램 로드" 클릭 후 클론한 폴더 선택

### API 키 설정

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 API 키 발급
2. 확장기능 설정 페이지에서 API 키 입력
3. [연결 테스트] 클릭하여 확인

## 사용 방법

1. Gmail에서 이메일 작성
2. 발송 전 **[검토하기]** 버튼 클릭
3. 모달에서 수정된 전체 이메일과 변경 요약 확인
4. **[반영하기]** 클릭 시 작성 창에 수정본 적용
5. Gmail의 **[보내기]** 버튼으로 발송

## 개인정보 처리

- 이메일 본문은 검토를 위해 Google Gemini API로 전송됩니다
- API 키만 로컬에 저장되며, 이메일 내용은 저장하지 않습니다
- 검토 이력은 메타데이터(타임스탬프, 카테고리 수)만 로컬에 보관합니다

## 기술 스택

- Chrome Extension Manifest V3
- Google Gemini API (Streaming)
- Shadow DOM (Gmail CSS 격리)
- 모달 다이얼로그 (수정본 + 변경 요약)
