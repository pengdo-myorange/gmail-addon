# Changelog

모든 주목할 만한 변경 사항이 이 파일에 기록됩니다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며,
버전 규칙은 [Semantic Versioning](https://semver.org/lang/ko/)을 준수합니다.

## [1.2.0] - 2026-04-18

### Added
- **되돌리기(Undo) 토스트**: 수정 반영 후 7초간 토스트 하단에 `되돌리기` 버튼을
  노출해, 한 번의 클릭으로 적용 직전 상태로 전체 복원 가능.
- **자동 롤백**: 수정 반영 과정에서 서명/인용 블록(`gmail_signature`,
  `gmail_quote` 등)이 의도치 않게 변경되면 자동으로 스냅샷을 복원하고,
  "서명·인용부가 영향을 받을 뻔해서 자동으로 되돌렸습니다" 토스트를 표시.
- **개별 반영 버튼**: 각 수정 항목에 `이것만 반영` 버튼을 추가해 한 건씩
  안전하게 적용할 수 있도록 통제 수단 제공. 반영된 항목은 시각적으로
  구분되며 체크박스/버튼이 비활성화됨.
- **안전 뱃지**: 모달 헤더 하단에 🔒 "서명 · 인용 · 링크 · 글꼴 · 색상은
  그대로 유지됩니다" 뱃지를 상시 노출.
- **수정 미리보기 diff 표시**: 미리보기 영역에서 수정된 구간을 `원본(취소선)
  → 수정본(하이라이트)` 형식으로 함께 표시하고, hover 시 원본 툴팁 제공.
- **서명/인용 경계 안내**: 미리보기 카드 하단에 서명·인용부는 검토 대상에서
  제외된다는 안내 문구 추가.

### Changed
- 모든 수정 항목이 반영되면 주 버튼이 `반영하기` → `완료 — 닫기` 로 자동 전환.
- 푸터 상태 문구를 선택/반영 건수 기준으로 업데이트.
- 기존의 `footer-hint` 문구는 더 눈에 띄는 `safety-badge`로 승격 이동.

### Fixed
- `_highlightCorrections`의 sentinel을 NUL(U+0000) 에서 Private Use Area
  (U+E000/U+E001) 문자로 교체. 일부 브라우저가 `textContent → innerHTML`
  경로에서 U+0000을 U+FFFD로 치환해 매칭이 실패하는 엣지 케이스 방지.
